# Student Outcome Platform

A React-based student learning outcomes management system built with Vite, featuring user authentication, course management, and outcome tracking.

## Features

- User registration and login (local storage)
- Role-based access (Student, Faculty, Admin)
- Course and learning outcomes management
- Submission tracking and marking
- Dashboard with charts and reports
- Responsive design with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:

```bash
npm install
```

### Running the Application

Start the development server:

```bash
npm run dev
```

The application will open at `http://localhost:5173`

### Building for Production

```bash
npm run build
```

### Linting

```bash
npm run lint
```

## Project Structure

- `src/components/` - Reusable UI components
- `src/pages/` - Page components
- `src/context/` - React contexts for state management
- `src/hooks/` - Custom React hooks
- `src/assets/` - Static assets and styles

## Technologies Used

- React 19
- Vite
- React Router DOM
- Tailwind CSS
- Framer Motion
